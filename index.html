Programação na web

Na Escola de Comercio de Lisboa , na discplina de Tecnologia 1 no 1ª ATIS, iremos aprender Programao web 😅 e abordaremos os principios basicos associados ao desenvolvimento para a Web, apenas do lado front-end, lado visual. Também sabemos que existe um lado , que é o lado do servidor, mas não iremos abordar esse lado.

Aprender-se-ão as seguintes tecnlogias 🌐:

  • HTML
  • CSS
  • JavaScript

De forma a consolidar todos os conhecimentos, serão feitos alguns laboratórios.
